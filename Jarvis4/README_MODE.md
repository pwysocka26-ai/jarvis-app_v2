Jarvis – tryby rozmowy

Nowe komendy (działają w locie, bez restartu):

/mode b2c   -> tryb domyślny (przyjaciel, ludzki, ciepły)
/mode pro   -> tryb PRO (konkretny, roboczy)
/mode       -> pokazuje aktualny tryb

To NIE używa zmiennych środowiskowych.
Tryb jest zapamiętywany w pamięci użytkownika.
