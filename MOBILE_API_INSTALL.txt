
JARVIS MOBILE API v1 — instalacja

Pliki w tej paczce:
- Jarvis4/app/api/mobile.py
- Jarvis4/app/schemas/mobile.py
- Jarvis4/app/services/mobile_service.py
- Jarvis4/app/mobile_main.py

Opcja A — szybki test bez ruszania obecnego main.py
1. Rozpakuj paczkę do katalogu projektu.
2. Uruchom:
   uvicorn app.mobile_main:app --reload --port 8011
3. Testuj:
   GET  /mobile/health
   GET  /mobile/today
   GET  /mobile/tomorrow
   POST /mobile/inbox
   POST /mobile/plan/tomorrow

Opcja B — wpięcie do głównego backendu
Dodaj do app/main.py:
   from app.api.mobile import router as mobile_router
   app.include_router(mobile_router)

Uwaga:
To jest Mobile API v1. Używa obecnych modułów Jarvisa jako silnika.
